# Examples (templates)

These scripts are meant to be **edited**: fill the small placeholders (data, formulas, settings) and run.

## PLboost (Power-Logit)

- `plboost_template.R`: boosted Power-Logit (PL) regression with `PowerLogit()` from `essentials/families_PL.R`.

## IPLboost (Inflated Power-Logit)

- `iplboost_template.R`: boosted Inflated Power-Logit (IPL) regression with `IPL()` from `essentials/families_IPL.R`.
